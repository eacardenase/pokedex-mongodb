# Setup Steps

1. Clone the repo
2. Execute the following command on the terminal

```
yarn install
```

3. Have Nest CLI already installed globally

```
npm i -g @nestjs/cli
```

4. Setup database

```
docker-compose up -d
```

## Stack

- MongoDB
- NestJS
